## 🐛 **TROUBLESHOOTING:**

### **"Microphone not working"**
```
Solution:
1. Check browser permissions (🔒 icon in address bar)
2. Click "Allow" for microphone access
3. Try refreshing the page
4. Use Chrome browser for best results
```

### **"Voice output not working"**
```
Solution:
1. Check speaker icon is 🔊 (not 🔇)
2. Increase system volume
3. Check browser isn't muted
4. Try clicking speaker button to toggle
```

### **"Speech not recognized accurately"**
```
Tips:
1. Speak clearly and slowly
2. Reduce background noise
3. Use short sentences
4. Try typing instead
```
